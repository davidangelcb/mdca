Acme Divi Modules adds some free extra modules and hacks to Elegant Themes Divi Builder

== Description ==

Acme Divi Modules adds some free extra modules and hacks to Elegant Themes Divi Builder

= Main Features =

* Change Project Slug: Replace default 'project' slug with custom slug
* Extend Page Builder to Custom Post Types: you can now easily extend the Page Builder to any Post Type.
* Change Divi image sized and ratio

= Acme Slide in Module *new* =

This module create an easy slide-in layer inside your website. You can use this module wherever you want inside your page: it will be displayed at the very top of your page. You can place a title, a custom text and a button.

= Acme Blog Module =

This module let you change the default sorting and adds a masonry layout

Features:

* Change the Order By default value;
* Switch between Ascending and Descending sort order;
* Added masonry.js support, you can choose the number of display columns;
* Size variation, you can show some item doble sized on a random/selction basis;
 

= Acme Portfolio Module =

This module hacks standard Divi behaviour, letting you use any post type and any taxonomy to display your feed.
You can also choose whether to show Excerpt.

Features

* Choose which post_type to display in portfolio;
* Customize default sort order and order by behaviour;


= Acme Fullwidth Portfolio Module =

This module hacks the fullwidth Divi Carousel, letting you use any post type and any taxonomy to display your feed.
You can also append the Excerpt and any custom field (strings only).
An option let you override default permalink with a custom url (from custom fields).
Column number will let you override the standard column number displayed

Features

* Choose which post_type to display in carousel;
* Customize the number of columns displayed;
* Customize default sort order and order by behaviour;
* Display extra custom fields
* Override default permalink with custom url

[ACME Media Kits](http://acmemk.com/en/acme-divi-modules "Acme Media Kits") is the home for this plugin.
== Installation ==


= From your WordPress dashboard =
1. Open Plugins Menu -> Add New;
2. Search for "Acme Divi Modules" inside the WordPress repository;
3. Activate it from the 'Plugins' menu in WordPress;

= From WordPress.org =
1. Download the zip archive;
2. Upload zip through Plugins -> Add New -> Upload Plugins;
3. Activate the plugin through the 'Plugins' menu in WordPress;

= Once Activated =
1. Open ACME -> Divi Modules;
2. Setup your Presets;
